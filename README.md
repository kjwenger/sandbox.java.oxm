# sandbox.java.oxm
Sandbox for all Java-related Object-to-XML-Mapping (OXM) experiments using a wide range of frameworks

## Prerequisites

- Java SDK
- Maven

```shell
sudo apt install openjdk-17-jdk maven
```

## Build

```shell
get clone --recursive https://github.com/kjwenger/sandbox.java.oxm.git
cd sandbox.java.oxm
mvn install
```
